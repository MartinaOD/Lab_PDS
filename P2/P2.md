## a) Lectura de la señal de audio

**Se lee el archivo de audio facilitado y se obtiene la señal x[n] y su frecuencia de muestreo fs1.**

**code**Matlab

```
[x, fs1] = audioread('PDS_P2_3A_LE2_G4.wav');   % x: NxC (C canales)
sound(x, fs1);                                  % Reproducir señal
disp(['Frecuencia de muestreo fs1: ', num2str(fs1), ' Hz']);
```

## b) Representación temporal de x(t)

**Representación de la señal en función del tiempo continuo aproximado a partir de las muestras x[n].**

**code**Matlab

```
N = length(x);                   % Número total de muestras
t = (0:N-1).' / fs1;             % Vector de tiempo en segundos

figure;                        
plot(t, x);                    
xlabel('Tiempo (s)');          
ylabel('Amplitud');            
title('x(t) a partir de x[n]');  
grid on;
```

**Explicación:** **Se representa la señal muestreada como función del tiempo continuo proyectando el índice de las muestras sobre la frecuencia de muestreo original fs1.**

## c) Cálculo de la señal y[n] (sin silencios)

**Cálculo de la señal y[n] eliminando los intervalos de reposo iniciales y finales donde la amplitud es menor a 0.01 V.**

**code**Matlab

```
umbral = 0.01;                  % Nivel mínimo de señal
idx = find(abs(x) >= umbral);   % Índices que superan el umbral

if isempty(idx)               
    error('No se ha encontrado señal por encima del umbral'); 
end                           

y = x(idx(1):idx(end));         % Recorte de la señal
ty = (0:length(y)-1).' / fs1;   % Nuevo vector de tiempo para y[n]

% Visualización de la señal recortada
figure;                       
plot(ty, y);                  
xlabel('Tiempo (s)');         
ylabel('Amplitud');            
title('y[n]: señal sin silencios iniciales y finales'); 
grid on;
```

**Explicación:** **El recorte elimina pausas innecesarias. Si existe ruido de fondo, se debería ajustar el umbral o utilizar métodos basados en la energía de la señal.**

## d) Función de Diezmado en Matlab

**Definición de la función para reducir la tasa de muestreo por un factor M.**

**code**Matlab

```
function y_n = Diezmador(x_n, M) 
    % Entradas: x_n (señal original), M (factor de diezmado)
    % Salida: y_n (señal diezmada)
    y_n = x_n(1:M:end); 
end
```

## e) Diezmado de y[n] con M = 2

**Obtención de la señal g[n] a partir de la señal y[n] previamente recortada.**

**code**Matlab

```
M = 2;                                   % Factor de diezmado
g = Diezmador(y, M);                     % Aplicación de la función
fs2_y = fs1 / M;                         % Nueva frecuencia de muestreo

disp(['fs1 = ' num2str(fs1) ' Hz, fs2_y = ' num2str(fs2_y) ' Hz']);
```

## f) Nueva frecuencia de muestreo fs2

**La nueva frecuencia de muestreo de la señal g[n] se calcula dividiendo la original por el factor M.**

* **Nueva frecuencia fs2:** **fs1 / M = 48000 Hz (si fs1 era 96kHz).**
* **Nueva frecuencia de x[n]:** **Si se aplicara el mismo proceso a x[n], su frecuencia también pasaría a ser fs2.**

## g) Justificación en el dominio del tiempo

**Comparación visual de las muestras originales y diezmadas para verificar la coincidencia de valores.**

**code**Matlab

```
tg = (0:length(g)-1).' / fs2_y;                
figure;                                        
plot(ty, y, 'b-o'); hold on;                     % Original en azul con círculos
plot(tg, g, 'r*');                               % Diezmada en rojo con asteriscos
xlim([0.5, 0.505]);                              % Zoom para apreciar las muestras
xlabel('Tiempo (s)'); ylabel('Amplitud');      
title('Justificación Tiempo: y[n] vs g[n]');   
legend('Original y[n]', 'Diezmada g[n]');      
grid on;
```

**Análisis:** **La gráfica muestra que el diezmado es correcto ya que las muestras de g[n] (asteriscos rojos) coinciden exactamente con una de cada dos muestras de y[n] (círculos azules). Al corregir el eje de tiempos con fs2, la forma de la onda se mantiene idéntica.**

## h) Justificación en el dominio de la frecuencia

**Análisis espectral utilizando frecuencias analógicas (Hz) para verificar la ausencia de distorsión.**

**code**Matlab

```
Ny = length(y); Ng = length(g);                
Y_mag = abs(fft(y))/Ny;                        
G_mag = abs(fft(g))/Ng;                        
fy = (0:Ny-1)*(fs1/Ny);                        
fg = (0:Ng-1)*(fs2_y/Ng);                      

figure;                                        
plot(fy(1:floor(Ny/2)), Y_mag(1:floor(Ny/2)),'b-o'); hold on; 
plot(fg(1:floor(Ng/2)), G_mag(1:floor(Ng/2)),'r*');         
xlim([1000, 1050]);                              % Zoom en un rango de frecuencias
xlabel('Frecuencia (Hz)'); ylabel('Magnitud');   
title('Justificación Frecuencia: y[n] vs g[n]'); 
legend('Original y[n]', 'Diezmada g[n]');      
grid on;
```

**Análisis:** **El espectro de g[n] coincide con el de y[n] dentro de la nueva banda de paso (0 a fs2/2 Hz). La coincidencia de los marcadores demuestra que la magnitud de las componentes se mantiene constante. El diezmado es válido porque la señal no tenía energía significativa por encima de la nueva frecuencia de Nyquist, evitando el solapamiento.**

## i) Análisis de Aliasing

**Pregunta:** **¿Qué pasaría al diezmar y[n] si su espectro tuviese energía entre fs1/2 y fs2/2? ¿Qué habría que hacer para evitarlo?**

**Respuesta:**
Si la señal tuviese energía en ese rango, se produciría el fenómeno de **aliasing** **(solapamiento espectral). Las componentes de alta frecuencia se "doblarían" sobre la banda de interés, distorsionando la señal resultante.**

**Para evitarlo, es necesario aplicar un** **filtro digital paso bajo (antialiasing)** **antes del proceso de diezmado. Este filtro debe tener una frecuencia de corte de fc = fs2/2 para asegurar que se elimine toda la energía que queda fuera de la nueva franja de Nyquist antes de descartar las muestras.**

Con la interpolación, modificamos la frecuencia original de muestreo incrementándolaa por un factor entero L.
Partiendo de la señal y[n] del bloque anterior, se realizan los siguientes apartados:

**a) Genera una  función de Matlab en la que se interpole una señal muestreada por un factor L dado. Indique los argumentos de entrada y salida de la función generada.**

```matlab
function y_l = interpolar(x, L)

% Generar función de Matlab para interpolar dado un facto L

% x = la entrada que se va a interpolar 
% L = factor de interpolación
% y_l = salida de la función, señal interpolada

% Generamos la nueva frecuencia de muestreo, partiendo de la Fs obtenida 
% con nuestra señal de audio, la cual es fs = 96000

% En primer lugar, descartamos aquellas L que no sean válidas:

if ((L<1) || (floor(L) ~= L)) % Verifico que L sea entero y >=1
    error('El factor L debe ser un entero mayor o igual a 1')
end

% Garantizado que la L con la que trabajamos  es válida...

% Un interpolador inserta ceros cada L muestras, ahora hemos de añadir
  
y_l = zeros(1, L*length(x));
y_l(1:L:end) = x;


end
```

**b) Realice la interpolación de y[n] con un coeficiente L = 2. La señal resultante será h[n].**

* Siendo la primera parte la de la obtención de y[n]:

```matlab
% >>>>> Teniendo lo realizado anteriormente para llegar a y[n]:

[x, fs1] = audioread("PDS_P2_3B_LE2_G4.wav");
sound(x,fs1);
disp(fs1);

% x, fs1 ya cargados
N = size(x,1);
t = (0:N-1)'/fs1;          % vector tiempo empezando en 0 (columnas)

% Representación
figure; plot(t, x);
xlabel('t (s)'); ylabel('Amplitude');
title('Representación temporal x_n'); grid on;

% x, fs1 ya cargados
thr = 0.01; % Umbral en voltios (según enunciado)

% Si x tiene dos canales, usar la envolvente (max absoluto entre canales)
if size(x,2) > 1
    env = max(abs(x), [], 2);
else
    env = abs(x);
end

% Encontrar índices activos (|x| >= thr)
active = find(env >= thr);

if isempty(active)
    error('No se encuentran muestras por encima del umbral.');
end

% Primer y último índice con sonido
i0 = active(1);
i1 = active(end);

% Extraer y[n]
y = x(i0:i1, :);   % conserva canales si existen
n_y = (0:size(y,1)-1)';  % índice n empezando en 0 para y[n]
t_y = n_y / fs1;         % vector tiempo empezando en t=0
```

* Una vez obtenida y[n], interpolamos con L=2:

```matlab
% >>>>> Teniendo la señal y[n]
L = 2;
h_n = interpolar(y, L); % Obtengo la nueva señal interpolada
```

**e) Indique cuánto debe valer la ganancia del filtro en la banda de paso.**

La ganancia del filtro en la banda de paso debe ser L para restaurar la energía que se perdió al insertar ceros, visible en cómo la amplitud de cada réplica tras interpolar baja a 1/L.
Al insertar ceros, no pierdes energía total (pues esta se conserva), pero sí "densidad de energía", porque ahora la señal ocupa más muestras; será el filtro el encargado de compensar eso multiplicando por L.

**f) Justifique la frecuencia de corte del filtro empleado.**

La frecuencia de corte tiene que ser del 1/(2*L) para evitar que las réplicas (originalmente centradas en 1 en frecuencia digital) se cuelen en la franja de Nyquist al comprimir el espectro 1/L.
Hemos de seguir respetando la periodicidad entre +-1/2.
