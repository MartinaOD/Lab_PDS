## BLOQUE 2: Implementación de un filtro FIR utilizando DFT

### Introducción teórica

La convolución lineal entre una señal de entrada **x[n]** y los coeficientes del filtro **b[n]** puede implementarse de forma eficiente utilizando la Transformada Discreta de Fourier (DFT) en lugar de cálculos directos en el dominio del tiempo.

La ventaja principal es la reducción de complejidad computacional:
- **Convolución directa:** O(N²)
- **Convolución mediante DFT:** O(N·log₂(N))

El algoritmo de **solape-almacenamiento** (overlap-add) consigue que las convoluciones circulares (propias de la DFT) sean equivalentes a convoluciones lineales mediante una gestión adecuada de las muestras de entrada y salida.

---

### **a) Implementación del algoritmo de solape-almacenamiento**

**Parámetros del algoritmo:**
- **L = 500:** Longitud de cada segmento de entrada
- **M:** Orden del filtro (longitud de los coeficientes b)
- **P = L + M - 1:** Longitud de la DFT y de cada segmento procesado

```matlab
% ========== IMPLEMENTACIÓN DEL ALGORITMO DE SOLAPE-ALMACENAMIENTO ==========

% Parámetros del algoritmo
L = 500;                    % Longitud de cada segmento de entrada
P = L + length(b) - 1;      % Longitud de la DFT (solape óptimo)
M_filt = length(b);         % Orden del filtro

% [COMPAÑERO 1 DEBE HABER CARGADO x y fs en el Bloque 1]
% Si no está cargado, cargar aquí:
% [x, fs] = audioread('nombre_archivo.wav');

N_total = length(x);        % Longitud total de la señal
num_segmentos = ceil(N_total / L);  % Número de segmentos a procesar

% Inicializar la señal de salida
y = zeros(N_total, 1);

% ========== BUCLE PRINCIPAL DEL ALGORITMO ==========

for k = 1:num_segmentos
    % 1. Extraer segmento de L muestras de la entrada
    inicio = (k-1) * L + 1;
    fin = min(k * L, N_total);
    x_seg = x(inicio:fin);
    
    % Rellenar con ceros si el último segmento es más corto que L
    if length(x_seg) < L
        x_seg = [x_seg; zeros(L - length(x_seg), 1)];
    end
    
    % 2. Calcular la convolución circular usando FFT
    % - Rellenar con ceros hasta longitud P
    X_seg = [x_seg; zeros(M_filt - 1, 1)];  % Relleno a P = L + M - 1
    B = [b; zeros(L, 1)];                    % Relleno a P
    
    % - Transformadas de Fourier
    X_fft = fft(X_seg, P);
    B_fft = fft(B, P);
    
    % - Multiplicación en el dominio de la frecuencia
    Y_fft = X_fft .* B_fft;
    
    % - Transformada inversa
    y_conv = ifft(Y_fft, P);
    
    % 3. Descartar los últimos (M-1) elementos (solape)
    y_seg = y_conv(1:L);
    
    % 4. Acumular los resultados en la señal de salida
    % (En los tramos solapados, sumar las contribuciones)
    fin_salida = min(inicio + L - 1, N_total);
    long_salida = fin_salida - inicio + 1;
    
    y(inicio:fin_salida) = y_seg(1:long_salida);
end

% ========== NORMALIZACIÓN (OPCIONAL) ==========
% Si es necesario normalizar la amplitud
y = real(y);  % Tomar la parte real (eliminar ruido numérico complejo)

% ========== REPRODUCCIÓN DE LA SEÑAL FILTRADA ==========
fprintf('Reproduciendo señal filtrada con overlap-add...\n');
sound(y, fs);

% ========== GRÁFICA: COMPARACIÓN TEMPORAL ==========
figure('Name', 'Señal Filtrada - Método Overlap-Add');
t = (0:length(y)-1) / fs;
plot(t, y, 'LineWidth', 1);
grid on;
title('Señal de salida y[n] - Algoritmo Overlap-Add (L=500)');
xlabel('Tiempo (s)');
ylabel('Amplitud');
```

---

### **b) Código comentado del algoritmo**

**Descripción de las líneas más significativas:**

| Línea | Significado |
|-------|-------------|
| `L = 500` | Longitud de cada segmento procesado (parámetro del algoritmo) |
| `P = L + M_filt - 1` | Longitud de la DFT (evita aliasing circular) |
| `for k = 1:num_segmentos` | Bucle sobre cada segmento de la señal |
| `X_seg = [x_seg; zeros(...)]` | Relleno con ceros hasta longitud P |
| `X_fft = fft(X_seg, P)` | Transformada discreta de Fourier del segmento |
| `Y_fft = X_fft .* B_fft` | Multiplicación en el dominio de la frecuencia (convolución circular) |
| `y_conv = ifft(Y_fft, P)` | Transformada inversa de Fourier |
| `y_seg = y_conv(1:L)` | Descartar los últimos (M-1) elementos del solape |
| `y = real(y)` | Extraer la parte real (eliminar ruido numérico complejo) |

### **c) Justificación del parámetro P**

```matlab
% Parámetro P (longitud de la DFT)
P = L + M - 1;

fprintf('Parámetro P (longitud de la DFT):\n');
fprintf('  L (longitud del segmento) = %d muestras\n', L);
fprintf('  M (orden del filtro) = %d\n', M_filt);
fprintf('  P = L + M - 1 = %d\n', P);
```

**Justificación matemática:**

> La convolución lineal entre un segmento de **L** muestras y un filtro de **M** coeficientes produce una salida de longitud **L + M - 1**. 
>
> Para que la convolución circular (mediante FFT) sea equivalente a la convolución lineal, la longitud de la FFT debe ser **al menos L + M - 1**. Por eficiencia computacional, **P es exactamente L + M - 1**, sin padding adicional.
>
> En este caso:
> - **L = 500** (muestras por segmento)
> - **M = ?** (número de coeficientes del filtro FIR)
> - **P = 500 + M - 1 = ?**
